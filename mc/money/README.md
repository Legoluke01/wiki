# Money

## How to get money

* Killing mobs
* Killing players
* Mining
* Fishing
* Completing achievements

## Where to spend money

* **`/shop`**
  * `/vehicleshop`
  * `/pshop`\(player shops\)
* **Lands/Nations**
  * Creating a Land/Nation
  * Claiming chunks
  * Paying upkeep/taxes
